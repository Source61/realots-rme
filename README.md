What is this?
=============

This is a map editor for OpenTibia, which is an open source implementation of the MMORPG Tibia with Realots SEC support.

I want to contribute
====================

Contributions are very welcome, if you would like to make any changes, fork this project or request commit access.

Bugs
======

Have you found a bug? Please create an issue in our [bug tracker](https://github.com/Source61/realots-rme/issues)

Other Applications
==========

* To host your MMORPG game server, you can use [Tibia Game](https://github.com/fusion32/tibia-game).
* To play your MMORPG game, you can use [OTClientV8](https://github.com/OTCv8/otcv8-dev)
* To map your MMORPG game, you can use this map editor.
